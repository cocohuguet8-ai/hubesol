## The difference at a glance

| | Present Perfect | Present Perfect Continuous |
|---|---|---|
| Focus | result / completed | activity / duration |
| Quantity | yes — I've read **3** books | no |
| How long | yes (with state verbs) | yes (for / since) |
| Example | **I've painted** the door. | **I've been painting** all morning. |

The key question: are you counting a **result**, or describing the **activity**? A result or a number takes the simple; the ongoing activity or its duration takes the continuous.

## Use the present perfect (simple) for…

- a finished result: **I've cleaned** the kitchen — it's spotless.
- a completed quantity: **She's sent** ten emails today.

> ⚠️ Typical error: ~~I've been reading three books this month.~~ → A number is a quantity, so use the simple: *I've read three books this month.*

## Use the present perfect continuous for…

- duration up to now: **I've been working** here for a year.
- a recent activity with a result now: **You've been crying** — your eyes are red.

## Common mistakes

**1. Continuous for a quantity.** *~~I've been writing five reports~~* → **I've written five reports.**

**2. Simple when the point is the activity.** For 'how long have you been doing X', use the continuous. *~~How long have you waited?~~* → **How long have you been waiting?**

**3. Stative verbs in the continuous.** *~~I've been having this car for years~~* → **I've had this car for years. Pro tip:** state verbs only take the simple.
