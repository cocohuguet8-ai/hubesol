## How to form the present perfect continuous

| Form | Structure | Example |
|---|---|---|
| Affirmative | subject + have/has + been + verb-**ing** | She **has been working**. |
| Negative | subject + haven't/hasn't + been + verb-ing | They **haven't been sleeping**. |
| Question | Have/Has + subject + been + verb-ing? | **Have** you **been waiting**? |

Use **have** with *I/you/we/they* and **has** with *he/she/it*. **Been** never changes, and the main verb takes **-ing**.

## When to use the present perfect continuous

### 1. How long, up to now

Often with *for* and *since*.

- **I've been learning** English **for** five years.
- **She's been living** here **since** 2019.

### 2. A recent activity with a result you can see now

- My eyes hurt — **I've been reading** all day.
- He's tired because **he's been working** in the garden.

> ⚠️ Typical error: ~~You've run — you're out of breath.~~ → For a recent activity explaining a result now, use the continuous: *You've been running — you're out of breath.*

## Common mistakes

**1. Using it for a finished quantity.** For a number or an amount, use the present perfect simple. *~~I've been reading three books~~* → **I've read three books.**

**2. Using it with stative verbs.** State verbs use the simple form. *~~I've been knowing him for years~~* → **I've known him for years. Pro tip:** know, want and like avoid the continuous.

**3. Leaving out 'been'.** The continuous always needs *been*. *~~I have working all day~~* → **I have been working all day.**
